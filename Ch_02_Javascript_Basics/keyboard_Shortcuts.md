# Keyboard Shortcuts

## VS Code Shortcuts

### General
| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + P` | Command Palette |
| `Ctrl + P` | Quick Open (Go to File) |
| `Ctrl + ,` | Open Settings |
| `Ctrl + K Ctrl + S` | Open Keyboard Shortcuts |

### Editing
| Shortcut | Action |
|----------|--------|
| `Ctrl + X` | Cut line (empty selection) |
| `Ctrl + C` | Copy line (empty selection) |
| `Ctrl + Shift + K` | Delete line |
| `Ctrl + Enter` | Insert line below |
| `Ctrl + Shift + Enter` | Insert line above |
| `Alt + Up` | Move line up |
| `Alt + Down` | Move line down |
| `Shift + Alt + Up` | Copy line up |
| `Shift + Alt + Down` | Copy line down |
| `Ctrl + D` | Add selection to next find match |
| `Ctrl + Shift + L` | Select all occurrences of current selection |
| `Ctrl + /` | Toggle line comment |
| `Shift + Alt + A` | Toggle block comment |
| `Ctrl + Shift + F` | Find in files |
| `Ctrl + H` | Find and replace |
| `Ctrl + Shift + H` | Replace in files |

### Navigation
| Shortcut | Action |
|----------|--------|
| `Ctrl + G` | Go to line |
| `Ctrl + Tab` | Switch between open files |
| `Ctrl + B` | Toggle sidebar |
| `Ctrl + J` | Toggle panel |
| `Ctrl + `` | Toggle integrated terminal |

### Code Editing
| Shortcut | Action |
|----------|--------|
| `Ctrl + Space` | Trigger suggestion |
| `Ctrl + Shift + Space` | Trigger parameter hints |
| `F12` | Go to definition |
| `Alt + F12` | Peek definition |
| `Shift + F12` | Find all references |
| `F2` | Rename symbol |
| `Ctrl + Shift + .` | Replace with next value |

## Playwright Test Specific (VS Code Extension)

| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + P` -> "Playwright" | Access Playwright commands |
| Click green arrow | Run single test |
| `npx playwright test` | Run all tests (terminal) |
| `npx playwright test --headed` | Run tests in headed mode |
| `npx playwright test --debug` | Run tests in debug mode |
